# PulseGrid AI

> **Real-Time Emergency Intelligence & Healthcare Coordination System**

AI-powered multi-agent platform designed to improve emergency response and healthcare coordination in rural and low-resource environments.

---

## 🚨 Problem Statement

Rural healthcare systems often face:
- Delayed emergency response
- Limited medical staff
- Language barriers
- Poor coordination between patients, ambulances, and healthcare providers
- Lack of real-time diagnostic assistance

These challenges can delay critical treatment and increase risk during emergencies.

---

## 💡 Solution

PulseGrid AI is a multi-agent emergency intelligence system that:
- Analyzes patient symptoms using AI
- Prioritizes emergency cases in real time
- Assists users through multilingual interaction
- Coordinates alerts for medical responders
- Provides fast medical summaries for healthcare workers

The platform is designed to support quick decision-making in low-resource healthcare environments.

## 🔄 System Workflow

1. User reports symptoms or emergency
2. AI agent analyzes the input
3. Priority agent classifies severity
4. Coordination agent alerts responders
5. Dashboard updates in real time

---

## 🏗️ Architecture

```text
[ User ]
   ↓
[ AI Triage Agent ]
   ↓
[ Priority & Decision Agent ]
   ↓
[ Emergency Coordination System ]
   ↓
[ Dashboard / Medical Response ]
```

## 🚀 Installation

### Clone Repository
```bash
git clone https://github.com/PromptPirates100/Hackathon.git
cd pulsegrid-ai
```

### Backend Setup
```bash
pip install -r requirements.txt
python app.py
```

### Frontend Setup
```bash
npm install
npm run dev
```

## 💻 Usage

1. Launch the application
